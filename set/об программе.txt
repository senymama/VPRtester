VPRteset Версия 3.8.3 (relese)
Данная версия является первым релизом

Для запуска программы для учителя необходимо открыть папку для учителя и запустить файл Создать тест
Для запуска программы для ученика необходимо открыть папку для ученика и запустить ярлык

По надобности можно переместить ярлык
ДРУГИЕ СИСТЕМНЫЕ ФАЙЛЫ ПЕРЕМЕЩАТЬ НЕЛЬЗЯ!

Интерфейс программы Создать тест:
в него вводится количеств вопросов которое будет в тесте(потом изменить нельзя)
далее нажать кнопку Передать

далее вводится вопрос и варианты ответов при помощи точек(справа от варианта ответа) выбирается правильный ответ, пока возможно только один.

После введения всех вопросов тест необходимо сохранить (в открывшемся окне), и далее передать данный файл ученику

Интерфейс Программы Пройти тест: 
после запуска программы предлагается выбрать режим чтения :
o = Открыть; c = вставить код ; e = выйти

Далее начинается тест

после теста появится окно с результатами теста (возможность сохранить пока нет)
